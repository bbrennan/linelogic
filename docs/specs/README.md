# Specs

This folder contains technical specifications and operator guides.

## Subfolders
- `deployment/` — POC deployment package, checklists, runbooks
- `training/` — training guides and training readiness docs
- `mlops/` — tracking, MLflow, operational architecture
- `roadmap/` — implementation roadmap and milestones
- `ui/` — dashboard and UI/UX design docs
- `storage/` — schema/table design and storage specs
- `setup/` — local/cloud setup docs
- `compliance/` — legal/compliance notes
- `model/` — model summaries and model-specific specs
